Welcome to the official repository for the **Christ Messengers Prophetic Ministry (CMPM)** website. This project is a modern, premium web application designed to reflect the ministry's "Divine Fire" mandate. It serves as a digital hub for members and visitors to connect with the ministry, access sermons, view upcoming events, and learn about the church's mission.

The website is built with a focus on **spiritual vibrancy**, **aesthetic excellence**, and **user experience**, featuring a custom "Divine Fire" design system with deep crimson and gold tones.

## âœ¨ Key Features

- **Premium "Divine Fire" Design**: A custom-tailored UI using deep red (`#be123c`) and royal gold colors to represent the Blood and Fire of the Spirit.
- **Dynamic Hero Section**: An immersive image slider with animated text overlays welcoming visitors.
- **Interactive Animations**:
  - **Parallax Scrolling**: Smooth visual depth effects on the Home page.
  - **Animated Background**: Subtle, floating church icons (Flame, Book, Cross) across all pages for a spiritual ambiance.
  - **Scroll-Aware Navbar**: Transitions from transparent to solid white as you scroll.
- **Comprehensive Content Sections**:
  - **Home**: Dynamic landing page with Events highlight, Latest Sermons, and "Prophetic Mandate" quote section.
  - **About Us**: detailed history, mission, vision, and leadership profiles.
  - **Ministries**: Information on various ministry arms like Zion Standard Academy.
  - **Contact**: Functional contact layout with location details.
  - **Giving**: Dedicated section/button for tithes and offerings.
- **Mobile Responsive**: Fully optimized layout for smartphones, tablets, and desktops.

## ğŸš€ Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/ex-commando/cmpm.git
   cd cmpm
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Start the Development Server**
   ```bash
   npm run dev
   ```
   The application will be available at `http://localhost:5173`.
